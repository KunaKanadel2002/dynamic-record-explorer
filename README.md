# Dynamic Record Explorer (Salesforce LWC)

A Lightning Web Component that allows you to:
- Select any Salesforce object
- Fetch records dynamically
- Apply client-side filters
- Search inside each record
- Expand to view full field details
- Enjoy premium iPhone-style glassmorphism UI

## Features
✔ Dynamic objects  
✔ Dynamic fields  
✔ Filters with AND logic  
✔ Per-record field search  
✔ Expand/Collapse cards  
✔ Glassmorphism UI theme  

## Installation
1. Clone this repo  
2. Deploy `/force-app` folder to your Salesforce org  
3. Add the LWC to a Lightning App or Record Page
